# Homebrew Tap for Hanthor Tools

This is a Homebrew tap containing formulas for hanthor CLI tools.

## Installation

```bash
brew tap hanthor/tap
brew install oramalama
```

## Available Formulas

- **oramalama** - Local AI coding assistant powered by RamaLama

## Usage

```bash
oramalama launch --tool pi
```
